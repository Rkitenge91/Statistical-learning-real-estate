# Statistical Learning Project: Real Estate Price Analysis

## Project Overview
This project applies statistical learning and machine learning techniques to analyze and predict real estate sales prices using the **Real Estate Sales dataset (2020–2024)**. The goal is to explore the relationships between property characteristics and sale prices, and to identify which features best predict property values.

The project focuses on building predictive models and evaluating their performance using modern statistical learning methods.

---

## Core Research Questions
1. Which property characteristics most strongly influence real estate sale prices?
2. Which statistical learning model best predicts real estate sale prices?
3. How do model performances compare across different machine learning techniques?

---

## Dataset
**Source:**  
Real Estate Sales Dataset (2020–Current)

Dataset link:  
https://catalog.data.gov/dataset/real-estate-sales-10012020-to-current

The dataset includes information such as:
- Sale price
- Property type
- Town / location
- Assessed value
- Sale date
- Other property-related characteristics

---

## Methods and Models
The following statistical learning methods will be explored:

- Linear Regression
- Ridge Regression
- Lasso Regression
- Elastic Net
- Tree-based models (if applicable)

Model performance will be evaluated using metrics such as:

- Mean Squared Error (MSE)
- Root Mean Squared Error (RMSE)
- Cross-validation
